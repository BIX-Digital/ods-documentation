Plugin installation

You can install the plugin manually by following the steps described in the "Installing files by file upload" section of the official documentation.

Change Plugin configurations

If you changed the default keycloak configurations, you need to update the parameters of the plugin.

  1. Stop the Confluence Server

  2. Locate the configuration file at <confluence-home>/classes/ods-keycloak.properties

  3. Insert the correct values for keycloak.baseURL, keycloak.realm and keycloak.clientId

  4. Restart the Confluence Server
